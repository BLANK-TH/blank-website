Tutorial - Collar Name Tag Generator

Config Description

All of the config boxes are already pre-filled with a default value to help you along.

Model Type

  • Base Model: Default base collar model
  • Shock Model: Default shock collar model
  • Custom: Specify a custom image to add a name tag to. If the collar location (in pixels) isn't the exact same as the base models, you'll need to manually edit it in the advanced section

Custom File

Specify a file if you selected the custom model type

Config

  • Name Tag Text: Text to display on the name tag
  • Font Size: Size of the font (Consolas Monospace) to use for the text, the more text you have, the smaller this needs to be (just go through trial and error)
  • X Padding: Padding between the text and background box borders on the X (width) axis
  • Y Padding: Padding between the text and background box borders on the Y (height) axis

Advanced Config

If you aren't using a custom model or have an explicit reason for adjusting these values, I recommend you don't.

The best way to get the pixel/location values for the image is to open it up in Microsoft Paint, hover over the point you want it to be with your mouse and look in the bottom left corner for the X and Y values.

  • Mid Origin: X value for the middle of the collar
  • Top Origin: Y value for the bottom of the collar/top of where you want the name tag to start
  • Max Height: Maximum height in pixels that the name tag can be
  • Max Width: Maximum width in pixels that the name tag can be

Result

After you click submit on the form, a new section will appear below it with a Heading called Result. It'll contain a cropped preview image of the collar model and name tag so you can edit the values to your liking and an Imgur link which you can use to attach the full texture.
